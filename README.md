# Электронный архив учебных материалов

Flask-приложение покрывает:
- базовая структура приложения и шаблоны страниц;
- аутентификация и авторизация пользователей;
- проверка прав доступа по ролям;
- CRUD для пользователей, категорий и материалов;
- загрузка, скачивание и удаление файлов;
- агрегирование данных и страница статистики.

## Отчет
- Отчет находится в папке `report` в виде LaTeX проекта и в корне репозитория в формате .pdf

## Стек
- Python
- Flask
- Flask-Login
- Flask-SQLAlchemy
- SQLite
- Bootstrap

## Как запустить
```bash
python -m venv ve
```

### Windows
```bash
ve\Scripts\activate
```

### Linux/macOS
```bash
source ve/bin/activate
```

Установка зависимостей:
```bash
pip install -r requirements.txt
```

Запуск:
```bash
python run.py
```

После первого запуска база создаётся автоматически.

## Тестовый администратор
- логин: `admin`
- пароль: `admin123`

## Роли
- `admin` — полный доступ;
- `teacher` — создание и редактирование своих материалов, загрузка и удаление файлов;
- `student` — просмотр материалов и скачивание файлов.
